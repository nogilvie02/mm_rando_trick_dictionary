# Majora's Mask Randomizer Trick Dictionary

A lightweight command-line utility to help MMR players know which techniques and glitches they can perform based on their current inventory.

## Features
- **Inventory Checklist**: Answer simple prompts to toggle which items you have in your inventory.
- **Trick Lookup**: Based on the inventory provided, include a list of all techniques and glitches that the player can perform.
- **Non-Area Specific**: These tricks are not area specific. The tricks are what techniques and glitches are possible given your inventory at a high-level.

## How To Run
To run the dictionary locally, make sure you have Python 3 installed, clone this repo, and run:
```bash
python main.py
```
